# Color Scheme Generator

Веб-приложение для генерации цветовых схем.

## Описание

Инструмент для создания гармоничных цветовых палитр, предназначенный для дизайнеров и разработчиков. Генерирует согласованные цветовые схемы для использования в веб-проектах, UI/UX дизайне и других творческих задачах.

## Возможности

- Генерация цветовых схем на основе базового цвета
- Подбор гармоничных комбинаций цветов
- Копирование значений цветов в различных форматах (HEX, RGB, HSL)
- Предпросмотр сгенерированной палитры
- Адаптивный интерфейс для работы на разных устройствах

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/IBlameVerieth/ColorScheme_generator.git
cd ColorScheme_generator
```

## Использование

Откройте файл `index.html` в браузере. Приложение не требует установки зависимостей или сборки.

Для локального сервера (опционально):

```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Затем откройте `http://localhost:8000` в браузере.
